# چک لیست تست همگام سازی دو دستگاه

1. روی Device A یک درس و یک فصل بسازید، سپس 50 سوال وارد کنید و مطمئن شوید محلی ذخیره شده است.
2. روی Device A وارد تب تنظیمات > اتصال گوگل درایو شوید و «همگام سازی با گوگل درایو» را بزنید تا فایل ابری ساخته/متصل شود.
3. روی Device B برنامه را تازه باز کنید، همان حساب گوگل را وصل کنید و «همگام سازی با گوگل درایو» را بزنید؛ باید همان 50 سوال بدون رفرش دستی نمایش داده شود.
4. روی Device B پنج سوال جدید اضافه کنید؛ پس از چند ثانیه Auto-Push باید اجرا شود.
5. روی Device A حداکثر طی 30 تا 60 ثانیه Auto-Pull باید اجرا شود و همان 5 سوال جدید را دریافت کند.

## نکته های عیب یابی

- در پنل «عیب یابی همگام سازی» وضعیت اتصال، توکن، File ID، زمان Pull/Push و نتیجه Merge را بررسی کنید.
- برای لاگ کامل، گزینه «لاگ دقیق کنسول» را فعال کنید.
- در صورت خطا، متن Stack را با دکمه «کپی خطا» کپی و بررسی کنید.
